# SCP - Inanimate Anomalies

SCP - Inanimate Anomalies is a heavily optimized SCP Minecraft plugin built specifically for Paper servers.<br>
#### NOTE: This plugin is incompatible with offline servers with no skins. Make sure to install [SkinsRestorer](https://modrinth.com/plugin/skinsrestorer) on offline servers in order for the plugin to work correctly.

## SCP - Inanimate Anomalies's features
This plugin adds 7 unique anomalies, all obtainable via the `/givescp` command. Each item grants its own unique abilities inspired by the SCP universe.

## Current Anomalous Items
Currently, there are 7 inanimate anomalies.

### Available Anomalies (via `/givescp`):
* **SCP-207 (The Cola)**
* **SCP-1853 (Performance Enhancer)**
* **SCP-330 (The Candy Bowl)**
* **SCP-018 (The Super Ball)**
* **SCP-500 (Panacea)**
* **SCP-034 (Ritual Obsidian Knife)**
* **The Jailbird** (Inspired by SCP: SL)

## Alpha Warhead Nuke
The plugin features a built-in extreme destruction tool inspired by the SCP universe. <br> 
It also has ray-tracing, allowing players to make warhead shelters.
- How to use:
  - place a lever or any sort of redstone as near as you can to it
  - activate the redstone

## SCP - Inanimate Anomalies Commands
`/alphawarhead [player]`<br>
Gives the player an alpha warhead nuke.

`/givescp <item> [player]`<br>
Gives a player a specific SCP item (e.g., panacea, jailbird).

## Plugin Permissions
`SCP.alphawarhead`<br>
Allows the use of the alphawarhead command. Default: op

`SCP.givescp`<br>
Allows the player to give SCP items. Default: op
